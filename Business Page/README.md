# Digital Agency Website

## What this project is

A clean, responsive website template for a digital agency that includes a homepage, About, Services, and Contact pages, ready to customize and deploy.

## Tech used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (vanilla)
- Google Fonts (Montserrat)

## Why it was built

To provide a lightweight, easy-to-customize starting point for agencies and small businesses who need a professional online presence quickly.

## What problem it solves

Reduces setup time and complexity for teams who want a polished, responsive site without learning build tools or frameworks.
